# Проект Блога

##Требуемый функционал:

1. Просмотр списка статей `(/posts) GET`
2. Просмотр списка собственных статей `(/authors/1/posts) GET`
3. Просмотр статьи и ленты комментариев к ней `(/posts/1/comments) GET`
4. Добавление, редактирование статей `(/posts) POST`
5. Удаление и публикация статей `(/posts/1) DELETE`
6. Добавление комментария к статье `(/posts/1/comments) POST`
7. Вход `(/login)`
8. Выход `(/logout)`

##Описание:
    
### Просмотр списка статей `[готово]`
Постраничный вывод списка опубликованных статей включает в себя: 
  1. Дату публикации 
  2. Автора 
  3. Заголовок 
  4. Тизер 
  5. Ссылку на просмотр статьи 
  6. Ссылки навигации по страницам списка


> `[готово]` Сортировка списка по дате публикации в обратном порядке. Ограничение количества записей на страницу - 5. Функционал общедоступен. 

### Просмотр списка собственных статей `[готово]`
Постраничный вывод списка всех собственных статей включает в себя: 
1. Дату создания
2. Заголовок
3. Тизер
4. Ссылку на просмотр статьи
5. Ссылки навигации по страницам списка

> `[готово]` Сортировка списка по дате создания в обратном порядке. Ограничение количества записей на страницу - 5. Функционал доступен только для авторизованного пользователя.


### Просмотр статьи и ленты комментариев к ней

Страница статьи включает в себя: 
  1. Дату создания, если статья не опубликована 
  2. Дату публикации, если статья опубликована 
  3. Автора, если статья не принадлежит текущему пользователю 
  4. Заголовок 
  5. Полный текст статьи 
  6. Ссылку на страницу публикации статьи, если функционал доступен 
  7. Ссылку на страницу удаления статьи, если функционал доступен 
  8. Ссылку на страницу добавления комментария, если функционал доступен 
  9. Список комментариев, включающий в себя текст комментария, информацию о дате размещения и авторе

> Сортировка списка комментариев по дате размещения в хронологическом порядке. Ограничение количества комментариев на страницу отсутствует. Функционал общедоступен.


### Добавление, редактирование статей `[готово]`

Требуемые входные параметры: 
  1. Заголовок - обязательное для заполнения поле, длина не более 255 
  2. Тизер - обязательное для заполнения поле, длина не более 255 
  3. Полный текст статьи - обязательное для заполнения поле, длина не более 3000

> Все поля - plaintext, без возможности размещения HTML и JS. При добавлении статья сохраняется в статусе "Черновик". При редактировании статус статьи не изменяется. В случае ввода невалидных данных выводятся сообщения об ошибках валидации. Редактирование доступно только для собственных статей в статусе "Черновик". Функционал доступен только для авторизованного пользователя.


### Удаление и публикация статей

> Функционал доступен только для авторизованного пользователя, для собственных статей в статусе "Черновик".

#### Добавление комментария к статье

Требуемые входные параметры: 
1. Текст комментария - обязательное для заполнения поле, длина не более 3000
2. Поле - plaintext, без возможности размещения HTML и JS. 

> В случае ввода невалидных данных выводятся сообщения об ошибках валидации. 

### Вход

Требуемые входные параметры: 
1. Имя - обязательное для заполнения поле 
2. Пароль - обязательное для заполнения поле

> При успешной проверке пары имя/пароль информация о текущем пользователе сохраняется в сессии. В обратном случае выводится сообщение об ошибке. Функционал доступен только для неавторизованного пользователя.

### Выход

> При выходе очищается информация о текущем пользователе. Функционал доступен только для авторизованного пользователя.
